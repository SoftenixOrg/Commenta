Sample Blog Post

This is a sample blog post to demonstrate the comments widget. The widget will appear below this content and allow users to login with Google and post comments with nested replies.

Features include:

  • Google OAuth authentication
  • Nested replies support
  • Light/dark theme toggle - Beta
  • Real-time comment posting
  • Like functionality - Beta
  • Edit and delete comments
  • Pagination for large comment threads
  • Anti-spam protection
  • XSS protection with content sanitization
